## Current position

Australian Home Collective has a live homepage, Guides index, ten populated category pages, 98 indexable guides and a complete public trust layer. Phase 1 established the publishing, metadata, structured-data, internal-link and commercial-boundary controls.

## Priority sequence

### 1. Maintain the existing library

- Use search and visitor evidence to find pages that need clearer answers, stronger practical detail or better discovery.
- Correct factual issues and weak sentence flow before adding volume.
- Use `Updated Month Year` only for substantive reader-facing revisions.
- Keep guide descriptions, canonicals, schema, links and images within the automated audit.

### 2. Complete the approved imagery work

- Finish only the replacements already specified in the visual audit.
- Maintain varied colour, materials and room settings.
- Reject branding, implausible cords or hoses, malformed products and decorative images that do not show the planning issue.

### 3. Prepare a controlled commercial pilot

- Apply to Commission Factory after this Phase 1 deployment is verified.
- Seek advertisers that fit existing guide topics.
- Keep the public catalogue empty until each product, destination, claim, disclosure and review date has formal approval.
- Start with a small number of contextual placements rather than a site-wide product rollout.

### 4. Add or expand content only when evidence supports it

- Do not create more generic storage variants while current guides cover the same decision.
- Prefer gaps in appliances, homewares and other higher-value home decisions when AHC can add verified practical value.
- A new article must pass the Editorial Decision Framework and must not begin with a product or affiliate opportunity.

## Deferred

- Named model recommendations and retailer or affiliate links remain blocked until commercial approval.
- No product database expansion is approved.
- No redesign or new category is planned.
- No fixed article-count target is used.

## Next task

After production verification, prepare the Commission Factory application using the approved description and checklist, without adding public commercial links.
